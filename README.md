--- Mini-arcade web ---
Par Cassandra Bélanger, Nesrine Haggui, Guy Junior Calvet, Émilie Harel et Adèle Dalle

--- Concept ---
Projet web permettant de jouer à des petits jeux directement dans le navigateur en incluant de la gestion de profils, des scores et un leaderboard

--- Technologies utilisées ---
Backend
  Node.js (v18+ recommandé)
  Express 4.17.1
  LowDB 1.0.0
  bcryptjs

Frontend
  HTML5 / CSS3 / JavaScript Vanilla (ES6+_
  Router maison basé sur le hash — router.js 
  ScoreService client-side pour envoyer les scores au serveur — scoreService.js 
  Mini-jeux dans /games :
    gameA.html
    gameA.js 
    gameA.css 

Outils de développement
  ESLint 9.39.1
